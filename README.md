# Phishing Awareness Presentation

## Overview

This project is a cybersecurity awareness presentation created as part of the CodeAlpha Cyber Security Internship. It explains phishing attacks, their types, warning signs, prevention methods, and best practices for staying safe online.

## Topics Covered

- Introduction to Phishing
- Types of Phishing
- Identifying Phishing Emails
- Fake Website Detection
- Social Engineering
- Real-World Example
- Prevention Best Practices
- Awareness Quiz

## Tools Used

- Microsoft PowerPoint
